pQCD ingredients¶

Running Coupling¶

Implementation: StrongCoupling.

We use perturbative QCD with the running coupling \(a_s(\mu_F^2) = \alpha_s(\mu_F^2)/(4\pi)\) given by [HRU+17] [LMMS16] [BCK17]

\[\frac{da_s}{d\ln\mu^2} = \beta(a_s) \ = - \sum\limits_{j=0} \beta_n a_s^{j+2}\]

Our main reference is [HRU+17].

The matching conditions between different number of flavors are given in [SS06] [CKS06].

At the moment we only implement an approximate solution to the RGE.

Splitting Functions¶

The Altarelli-Parisi splitting kernels can be expanded in powers of the strong coupling \(a_s(\mu^2)\) and are given by [MVV04] [VMV04]

\[\begin{split}\mathbf{P}(x,\mu^2) &= \sum\limits_{j=0} a_s^{j+1}(\mu^2) \mathbf{P}^{(j)}(x) \\ \tilde{\gamma}^{(j)}(N) &= -\mathcal{M}[\mathbf{P}^{(j)}(x)](N)\end{split}\]

Note the additional minus in the definition of \(\tilde \gamma\).
