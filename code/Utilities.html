Utility Classes¶

Apart from the operator classes, eko also provides some utility classes which are e.g. used in yadism

  • eko.thresholds.ThresholdsConfig

    • Implementation of the flavor number scheme and the quark thresholds both for the eko.strong_coupling.StrongCoupling and the operators

ThresholdConfig parameters:

  • str FNS active flavor number scheme. Possible values are FFNS, representing a FFNS, or, ZM-VFNS, FONLL-A, FONLL-A’, representing a VFNS; Required

  • float Q0 reference scale from which to start; Required

  • int NfFF number of fixed flavors if applicable; Required if in FFNS, ignored otherwise

  • float mc charm mass in GeV; Required if in VFNS, ignored otherwise

  • float mb bottom mass in GeV; Required if in VFNS, ignored otherwise

  • float mt top mass in GeV; Required if in VFNS, ignored otherwise

  • eko.strong_coupling.StrongCoupling Implementation of the Strong Coupling

  • eko.interpolation.InterpolatorDispatcher Implementation of the Interpolation
