EKO documentation
=================

EKO solves the DGLAP equations in Mellin space and produces evolution kernel operators.
It is thus PDF-independet and the operators can be computed for a given configuration
once and be reused after.

This documentation is organised in three sections: first, an quick overview on the theoretical ingredients is given,
second, an overview of the implementation details, and, last, the available indices.
